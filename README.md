# Forest Fire
 
"Bringing Clarity to the Canopy: Innovating Forest Fire Detection."

Forest Fire Detection powered by deep learning revolutionizes the way we safeguard our natural ecosystems. By harnessing the intelligence of artificial neural networks, this innovative solution provides early detection of forest fires, enabling rapid response and mitigation efforts. Through sophisticated analysis of thermal imaging data and environmental factors, it offers unparalleled accuracy, helping to minimize the devastating impact of wildfires on our forests, wildlife, and communities.

# How to Run 

-> Downlaod the files.
-> Install the Required packages
-> Click Run
-> Show any image of a forest fire to the webcam
-> It will turn greyscale if fire is detected
